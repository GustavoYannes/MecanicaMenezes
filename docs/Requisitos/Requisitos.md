| ID   | Requisito             | Descrição                                                                                  | Prioridade / Complexidade |
| ---- | --------------------- | ------------------------------------------------------------------------------------------ | ------------------------- |
| RF01 | Cadastro de Entrada   | O usuário deve cadastrar a entrada de um veículo na oficina.                               | Alta / Alta               |
| RF02 | Login do Usuário      | O gerente/mecânico deve conseguir realizar login no sistema.                               | Alta / Baixa              |
| RF03 | Dashboard de Veículos | O usuário deve visualizar todos os veículos atualmente na oficina.                         | Alta / Média              |
| RF04 | Cadastro de Cliente   | O cliente deve ser cadastrado automaticamente ao registrar uma entrada, caso não exista.   | Alta / Média              |
| RF05 | Cadastro de Veículo   | O veículo deve ser cadastrado automaticamente ao registrar uma entrada, caso não exista.   | Alta / Média              |
| RF06 | Controle de Status    | O veículo deve possuir status: “Em espera”, “Em andamento” e “Concluído”.                  | Alta / Média              |
| RF07 | Validação de Entrada  | Um veículo só pode dar nova entrada se estiver com status “Concluído”.                     | Alta / Média              |
| RF08 | Adição de Serviços    | Deve ser possível adicionar serviços a um veículo enquanto estiver na oficina.             | Alta / Alta               |
| RF09 | Cálculo de Orçamento  | O sistema deve somar os serviços e gerar o valor total do orçamento.                       | Alta / Média              |
| RF10 | Geração de PDF        | O sistema deve gerar um orçamento em PDF para download.                                    | Alta / Alta               |
| RF11 | Histórico de Clientes | Deve existir uma tela com todos os clientes e acesso ao histórico de entradas.             | Média / Média             |
| RF12 | Histórico de Veículos | Deve existir uma tela com todos os veículos e acesso ao histórico de entradas.             | Média / Média             |
| RF13 | Consulta de Cliente   | O usuário deve visualizar detalhes completos de cliente                                    | Média / Média             |
| RF14 | Consulta de Veiculo   | O usuário deve visualizar detalhes completos de Veiculos                                   | Média / Média             |
| RF15 | Dashboard do Sistema  | Devera ter um dashboard com Dados do sistema                                               | Baixa / Média             |
